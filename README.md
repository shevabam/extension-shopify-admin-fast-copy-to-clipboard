# Shopify Admin Fast Copy to Clipboard

This Chrome extension **adds a small Copy button** next to common fields in the Shopify Admin interface. With a single click, you can copy field values (like product title, SKUs, handles, price, weight, etc.) straight to your clipboard.

![](docs/shopify-adminn-fast-copy-to-clipboard_0.jpg)

## Why use this extension?

- **Save time**: Copy frequently used values in one click instead of selecting text and pressing shortcuts
- **Subtle and accessible**: Buttons sit next to fields, highlight copied fields, and show a small tooltip confirmation
- **Toggle on/off**: Quickly enable or disable from the popup without reloading the page
- **Light/Dark theme**: Choose between light and dark theme in the popup

[![Watch demo](docs/watch-demo.png)](https://www.youtube.com/watch?v=-NUEQ5Ul_r0)


## Installation

Download the extension from the [**Chrome Web Store**](https://chromewebstore.google.com/detail/shopify-admin-copy-to-cli/gmmdkkjncbflfbkdgcklogfnoajlkjjp?authuser=0&hl=fr) or **Microsoft Edge Store** (coming soon!)

After installing the extension, pin it to your browser:

![](docs/shopify-adminn-fast-copy-to-clipboard_1.png)


## Usage

1. Navigate to your Shopify Admin (`https://admin.shopify.com/`) and go to a product page
2. Look for small copy buttons next to configured fields
3. Click a button to copy the field's value. A tooltip confirms success, and the field briefly highlights
4. Use the extension popup to enable/disable the feature or change the popup theme

## Privacy

No external requests or analytics. The extension operates entirely client-side in your browser. Settings are kept in Chrome's synced storage.
